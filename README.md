Benvenuti nella repository ufficiale del sito **Area 51**, il punto di riferimento per lo street food galattico a Vimercate. Questo progetto è una Single Page Application (SPA) moderna, veloce e ottimizzata per la conversione dei clienti.

---

## 📱 Panoramica del Progetto

Il sito è stato progettato per offrire un'esperienza utente fluida (UX), permettendo ai clienti di passare dalla consultazione del menu alla prenotazione in meno di 30 secondi.

### 🚀 Funzionalità Chiave
| Feature | Descrizione |
| :--- | :--- |
| **Menu Interattivo** | Filtri rapidi per Panini, Pinse, Fritti e Drink. |
| **Booking Engine** | Form di prenotazione personalizzato con validazione dati. |
| **Responsive Design** | Layout adattivo per smartphone, tablet e desktop. |
| **Direct Contact** | Pulsanti rapidi per WhatsApp, Chiamate e Google Maps. |

---

## 🛠️ Stack Tecnologico

> [!NOTE]
> Il progetto è costruito con tecnologie standard del web per garantire massime prestazioni e indicizzazione SEO.

* **Frontend:** HTML5, CSS3 (Custom Grid & Flexbox).
* **Interattività:** JavaScript Vanilla (ES6+) per la gestione del DOM.
* **Hosting:** Netlify con Continuous Deployment.
* **Iconografia:** FontAwesome & Custom SVG.

---

## 📂 Struttura delle Sezioni

Il layout è diviso in blocchi logici per guidare l'utente:

1.  **Hero Section:** Impatto visivo immediato con CTA (*Call to Action*).
2.  **Menu Digitale:** Presentazione dei piatti con nomi a tema spaziale (es. *Saturno Burger*).
3.  **Specialità:** Box evidenziati per Arrosticini e Alette.
4.  **Eventi:** Sezione dedicata alla musica dal vivo e serate a tema.
5.  **Modulo Prenotazione:** Sistema di invio richieste per riservare un tavolo.
